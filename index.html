El diseño web responsivo es una técnica que permite que un sitio web se adapte automáticamente a distintos dispositivos como celulares, tablets y computadoras, mejorando la experiencia del usuario.

Viewport
Es el área visible de una página web en la pantalla del dispositivo.

Papel del viewport
Permite que el contenido se ajuste correctamente a diferentes tamaños de pantalla.

Configuración
Se define con la etiqueta <meta name="viewport">.

CSS3 Media Queries

Las media queries permiten adaptar el diseño de un sitio web según el tamaño de la pantalla del dispositivo.

Aplicación práctica

Se utilizan para modificar columnas, textos y elementos, creando diseños web responsivos.
